# Jellyfish Image Classification Web App

This is a simple deep learning project that classifies jellyfish images using a CNN model and a Flask web interface.

#### 🚀 Features

Image classification of jellyfish species
Trained CNN model using TensorFlow/Keras (MobileNetV2)
Flask web app for uploading images and getting predictions

#### 🧠 Model

The model is based on MobileNetV2 with a custom classification head for 6 jellyfish classes.

#### 🖼️ Classes

Moon jellyfish
Barrel jellyfish
Blue jellyfish
Compass jellyfish
Lion’s mane jellyfish
Mauve stinger jellyfish

#### ⚙️ Tech Stack

Python, TensorFlow, Keras, Flask, NumPy, Pillow

#### How to run

`pip install -r requirements.txt
python app.py`
